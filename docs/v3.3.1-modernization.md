> **Статус**: IN PROGRESS | **Дата**: 2026-04-14

## Результаты глобального аудита v3.3.0

### Скриншот-анализ (Obsidian live testing)

| Скриншот | Наблюдение | Корневая причина |
|----------|-----------|-----------------|
| Диалог "Добавить новый вид" | Database в списке рядом с Table — OK | — |
| Database View + DataTable | Столбцы визуально смещены; checkbox/color поля слишком широкие | scrollbar-gutter + px-flex без адаптивных defaults |
| FormulaBar Visual mode | IF-блок рендерится; chart виджеты "не настроены" | Пустой config → `asChartConfig()` returns null → fallback |

### Критические проблемы (P0)

1. **i18n corruption** — `common` секция в `ru.json` (16 ключей) и `uk.json` (15+ ключей) обрезана (пропущена первая буква каждого значения)
2. **Смещение столбцов таблицы** — virtual scroll `overflow-y: auto` сужает контейнер на ширину скроллбара; aggregation row с отдельным `overflow-x: auto` не синхронизирован

### Высокий приоритет (P1)

3. **Адаптивные defaults ширины столбцов** — boolean: 80px вместо 180px, date: 120px, number: 100px
4. **Aggregation picker** — нет UI для выбора функции агрегации в футере
5. **Error boundary** — crash одного виджета ломает весь DatabaseView
6. **Пропущенные i18n ключи** — `open-in-tab`, `open-in-window` отсутствуют в uk.json и zh-CN.json
7. **Chart fallback** — вместо "не настроен" показывать inline wizard настройки

### Средний приоритет (P2)

8. Visual formula editor не поддерживает операторы (`a > b`)
9. Free layout — нет resize handles и position persistence
10. Матрёшка-архитектура (`@container` queries) не применена к DataGrid
11. Контекстные меню DataGrid — legacy кастомные, не Obsidian `Menu` API

### Низкий приоритет (P3)

12. Demo vault полностью пуст
13. Table view deprecation strategy (banner/auto-redirect)
14. User guide не описывает Database View

## Gap Analysis: Спецификация vs Реализация

### Полностью реализовано ✅
- Widget infrastructure (6 типов, registry, host, toolbar)
- Transform Pipeline (6 steps, executor, cache, timeout, ReDoS protection)
- Formula Engine (52 функции computed mode)
- SVG Charts (9 sub-types, pure SVG, theme-aware)
- Stats Widget (19 aggregation functions, auto-detection)
- Relation Resolver + Rollup Engine
- Config migration (TableConfig → DatabaseViewConfig)
- Virtual scroll (>100 rows)
- Widget templates (3 presets)
- DnD widget reorder (stack mode)
- Mobile stack-only layout
- i18n (4 locales, ~90 DB keys) — кроме corrupted common
- Conditional formatting
- Row grouping

### Частично реализовано ⚠️
- **FormulaBar Visual mode** — функции OK, операторы не поддержаны
- **Aggregation Row** — вычисления OK, UI picker для выбора функции absent
- **Free layout** — CSS Grid 12-col работает, но позиции не персистируются, нет resize handles
- **Keyboard navigation** — ariaGrid/ariaWidget helpers есть, grid Tab/Arrow частично
- **Context menus** — DataGrid legacy; spec требует Obsidian Menu API

### Не реализовано ❌
- **Aggregation picker UI** (`[▼]` dropdown в футере)
- **Error boundary** (widget crash protection)
- **Chart inline wizard** (setup при пустом config)
- **Container queries** (Матрёшка для DataGrid)
- **Demo vault** (пуст)
- **User guide** Database View section
